## Task Goal

Stand up the `skills/xlsx/scripts/xlsx_read/` package as an
**importable Python module** whose **public contract is complete**
(every dataclass, enum, exception, and method signature in
ARCHITECTURE §5.1 is present) but whose **logic is stubbed**
(`NotImplementedError`, `return None`, hardcoded sentinels). Land
the `ruff` banned-api toolchain so the closed-API gate is live
**from this commit forward** (D5). Add one stub E2E test that
imports the package, instantiates a `WorkbookReader`, and asserts
the sentinel return values — Red→Green per `tdd-stub-first §1.4`.

End-state:
- `python3 -c "import xlsx_read"` succeeds.
- `ruff check skills/xlsx/scripts` exits 0 (no banned-api violations
  in tracked code).
- `skills/xlsx/scripts/xlsx_read/tests/test_smoke_stub.py` passes,
  asserting hardcoded stub returns.
- `skills/xlsx/.AGENTS.md` and `skills/xlsx/SKILL.md §10` updated.
- 12-line `diff -q` cross-skill silent gate is silent.

   `issubclass(EncryptedWorkbookError, RuntimeError)`,
   `issubclass(MacroEnabledWarning, UserWarning)`,
   `issubclass(OverlappingMerges, RuntimeError)`,
   `issubclass(AmbiguousHeaderBoundary, UserWarning)`,
   `issubclass(SheetNotFound, KeyError)`.
3. **TC-UNIT-03 (dataclasses frozen):** Assert
   `dataclasses.is_dataclass(SheetInfo)` and that mutating a public

   `_sheets.enumerate_sheets(None)`, `_merges.parse_merges(None)`,
