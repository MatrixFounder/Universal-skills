# Style: Crypto / Web3 (Telegram Focus)

**Tone:** Fast, skeptical but excited, insider.
**Target Audience:** Degen traders, investors, builders.

## Keywords to Avoid (The "Npc" Words)
*   **Avoid:** "Revolutionizing finance," "transforming the landscape," "game-changer," "unlocking potential."
*   **Why:** These are VC/Press Relesase buzzwords. Real users ignore them.

## Preferred Vocabulary (The "Alpha" Words)
*   Instead of "This project allows users to...", use **"Enables LPs to..."**
*   Use standard acronyms: **TVL, TPS, L2, zk-rollup, perp**.
*   Use "narrative" instead of "trend".
