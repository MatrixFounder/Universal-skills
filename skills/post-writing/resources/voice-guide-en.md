## Anti-Patterns

**DO NOT teach (show instead):**
```
// Bad
"You need to understand that AI agents are the future."

// Good
"I still believe agents are the next big step after ChatGPT. So I decided to build a platform."
```

**DO NOT use filler:**
```
// Bad
"In today's world of rapidly evolving technology, artificial intelligence plays an increasingly important role..."

// Good
"Every other post is about the agent revolution."
```

**DO NOT use jargon without context:**
```
// Bad
"Prompt engineering and RAG are critical for the tokenizer pipeline."

// Good
"Prompt engineering is the skill of talking to AI so it actually understands what you need."
```
