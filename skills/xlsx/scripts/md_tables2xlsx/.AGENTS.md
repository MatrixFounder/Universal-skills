- **ARCH M1** — `tables.py` module-level `_HTML_PARSER = lxml.html.HTMLParser(no_network=True, huge_tree=False, recover=True)` singleton. Test: `test_html_billion_laughs_neutered` exercises a 100-level nested-entity payload, asserts wall-clock ≤ 1s + content < 1KB.

Test: `test_convert_md_tables_to_xlsx_atomic_token_protection` (`sheet_prefix="--evil-flag-attempt"` becomes a single token).

Tests: `test_header_fill_matches_csv2xlsx` + `test_header_fill_matches_json2xlsx` + `test_max_col_width_matches_csv2xlsx`.
