# Directory: skills/xlsx/scripts/

## Purpose

Single-file Python CLIs for the xlsx skill. Every script in this
directory takes a `.xlsx`/`.xlsm` path on argv and produces a
deterministic side-effect (a new file, a recalculated workbook, a
validation report, or — new for xlsx-6 — an inserted comment). All
scripts share three cross-skill contracts:

- **cross-3** (encryption): reader scripts call `office._encryption.assert_not_encrypted` to fail fast on legacy CFB / encrypted inputs (exit 3).
- **cross-4** (macros): reader/writer scripts emit a stderr warning via `office._macros.warn_if_macros_will_be_dropped` when `.xlsm → .xlsx` would silently strip VBA.
- **cross-5** (json-errors): every script exposes `--json-errors` to emit failures as a single-line JSON envelope (`{v, error, code, type?, details?}`) on stderr — see `_errors.py`.
- **cross-7-H1** (writer scripts): refuse same-path INPUT/OUTPUT (incl. via symlinks resolved through `Path.resolve()`) with exit 6 `SelfOverwriteRefused`.

The `office/`, `_errors.py`, `_soffice.py`, and `office_passwd.py`
modules are byte-identical to the docx master per CLAUDE.md §2 — do
NOT edit them here.

## Files

### csv2xlsx.py
**Purpose:** CSV/TSV → styled `.xlsx` (bold header, frozen first row, auto-filter, leading-zero preservation).

### xlsx_recalc.py
**Purpose:** Force formula recalculation via headless LibreOffice; optional `--scan-errors`.

### xlsx_validate.py
**Purpose:** Fast scan for cached formula errors (`#REF!`, `#DIV/0!`, etc.) without recalc. `--fail-empty` catches the "formulas never recalculated" case.

### xlsx_add_chart.py
**Purpose:** Add a bar/line/pie chart over a cell range; chart stays editable in Excel/LibreOffice.

### xlsx_add_comment.py *(xlsx-6, v1)*

**Purpose:** Insert a Microsoft Excel comment into a target cell.
Single-cell mode (`--cell REF --author NAME --text MSG`) or batch mode
(`--batch path-or-dash`, auto-detects flat-array vs xlsx-7 findings
envelope). Threaded mode (`--threaded`, Q7 closure: writes BOTH a
legacy stub AND a threadedComment + personList per Excel-365 fidelity)
opt-in.

**Status:** v1 — full pipeline implemented (no stubs). Test surface:
75 unit tests + 112 e2e tests + 5 canonical-XML goldens, all green.
`office/validate.py` clean on every produced workbook.
`validate_skill.py skills/xlsx` exit 0.

**Functional regions** (file is organised into `# region — Fx` blocks):
- F1 — argparse surface (`build_parser`, `_validate_args`).
- F2 — cell-syntax parser + sheet resolver (`parse_cell_syntax`,
  `resolve_sheet`; M2 first-VISIBLE / M3 case-sensitive).
- F3 — batch loader (`load_batch`, `BatchRow`; flat-array + xlsx-7
  envelope auto-detect; m-4 8 MiB pre-parse cap).
- F4 — OOXML editor (`scan_idmap_used` list-aware per M-1,
  `scan_spid_used`, `next_part_counter`; `ensure_*` get-or-create
  helpers for legacy / threaded / person / VML parts;
  `add_legacy_comment` / `add_threaded_comment` / `add_person` /
  `add_vml_shape`).
- F5 — merged-cell resolver + duplicate-cell matrix
  (`resolve_merged_target`, `detect_existing_comment_state`,
  `_enforce_duplicate_matrix`; ARCH §6.1 4-row matrix).
- F6 — orchestration (`main`, `single_cell_main`, `batch_main`,
  `_post_pack_validate` env-gated guard).

**Local exception classes** (caught in `main()`, routed through
`_errors.report_error`):
- exit 1: `OutputIntegrityFailure`, `MalformedVml`, generic I/O.
- exit 2: `UsageError`, `SheetNotFound`, `NoVisibleSheet`,
  `InvalidCellRef`, `MergedCellTarget`, `EmptyCommentBody`,
  `InvalidBatchInput`, `BatchTooLarge`, `MissingDefaultAuthor`,
  `DuplicateLegacyComment`, `DuplicateThreadedComment` (M-2).
- exit 3: `EncryptedFileError` (cross-3).
- exit 6: `SelfOverwriteRefused` (cross-7 H1).

**Constants:**
- `DEFAULT_VML_ANCHOR = "3, 15, 0, 5, 5, 31, 4, 8"` — Excel's default
  offset; locked by R9.c (no custom positioning in v1).
- `_BATCH_CAP_BYTES = 8 * 1024 * 1024` — pre-parse cap on `--batch`
  input; m-4 boundary `read(8 * MiB + 1)`.
- `XLSX_ADD_COMMENT_POST_VALIDATE` (env var, truthy 1/true/yes/on) —
  enables the in-process post-pack `office/validate.py` guard. Default
  off; CI / E2E set it.
- Namespace constants for `ss`, `r`, `pr`, `ct`, `v`, `o`, `x`,
  `threaded` + content-type strings for `comments`, `vmlDrawing`,
  `threadedComments`, `personList`.

**Honest scope** (R9.a–R9.g) locked in
`tests/test_xlsx_add_comment.py::TestHonestScope`: no `parentId`
(reply threads), plain-text body only, default VML anchor only,
goldens are agent-output-only (Excel may mutate on save), UUIDv4
`<threadedComment id>` is non-deterministic by design, no
`--default-initials`, no `--unpacked-dir`. See
`references/comments-and-threads.md` §4 for the full list.

### preview.py
**Purpose:** Universal `INPUT → PNG-grid` renderer for `.xlsx`/`.xlsm`/`.docx`/`.pptx`/`.pdf`. **Byte-identical** across docx/xlsx/pptx/pdf — do not edit here.

### office_passwd.py
**Purpose:** Set/remove/detect password protection on `.xlsx`/`.docx`/`.pptx` via msoffcrypto-tool. **Byte-identical** across docx/xlsx/pptx — do not edit here.

### _errors.py
**Purpose:** `--json-errors` envelope helper (schema `v=1`). **Byte-identical** across all four office skills — do not edit here.

### _soffice.py
**Purpose:** LibreOffice subprocess wrapper. **Byte-identical** across all four office skills — do not edit here.

### office/ (subdirectory)
**Purpose:** OOXML unpack/pack/validate, validators, helpers, encryption / macros pre-flight, AF_UNIX shim. **Byte-identical** to `skills/docx/scripts/office/` — see CLAUDE.md §2.

### tests/
**Purpose:** E2E (`test_e2e.sh` — 112 passing tests, runs all xlsx skill
CLIs end-to-end including the xlsx-6 v1 surface + 5 canonical-XML
goldens) and Python unit tests (`test_xlsx_add_comment.py` — 75 tests,
0 skipped; `office/tests/` — 43 tests). Helpers: `_golden_diff.py`
(c14n + volatile-attribute mask), `regenerate_synthetic_inputs.py`
(rebuilds `golden/inputs/` from synthetic specs).

### requirements.txt
**Purpose:** `openpyxl`, `lxml`, `defusedxml`, `pandas`, `msoffcrypto-tool`, `pypdf` (required by some preview paths). xlsx-6 introduces NO new dependencies — uses lxml + stdlib only.

### install.sh
**Purpose:** Bootstrap `scripts/.venv/` and pip-install requirements. Idempotent.
