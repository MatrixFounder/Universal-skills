# Directory: skills/xlsx/scripts/

## Purpose

Single-file Python CLIs for the xlsx skill. Every script in this
directory takes a `.xlsx`/`.xlsm` path on argv and produces a
deterministic side-effect (a new file, a recalculated workbook, a
validation report, or — new for xlsx-6 — an inserted comment). All
scripts share three cross-skill contracts:

- **cross-3** (encryption): reader scripts call `office._encryption.assert_not_encrypted` to fail fast on legacy CFB / encrypted inputs (exit 3).
- **cross-4** (macros): reader/writer scripts emit a stderr warning via `office._macros.warn_if_macros_will_be_dropped` when `.xlsm → .xlsx` would silently strip VBA.
- **cross-5** (json-errors): every script exposes `--json-errors` to emit failures as a single-line JSON envelope (`{v, error, code, type?, details?}`) on stderr — see `_errors.py`.
- **cross-7-H1** (writer scripts): refuse same-path INPUT/OUTPUT (incl. via symlinks resolved through `Path.resolve()`) with exit 6 `SelfOverwriteRefused`.

The `office/`, `_errors.py`, `_soffice.py`, and `office_passwd.py`
modules are byte-identical to the docx master per CLAUDE.md §2 — do
NOT edit them here.

## Files

### csv2xlsx.py
**Purpose:** CSV/TSV → styled `.xlsx` (bold header, frozen first row, auto-filter, leading-zero preservation).

### xlsx_recalc.py
**Purpose:** Force formula recalculation via headless LibreOffice; optional `--scan-errors`.

### xlsx_validate.py
**Purpose:** Fast scan for cached formula errors (`#REF!`, `#DIV/0!`, etc.) without recalc. `--fail-empty` catches the "formulas never recalculated" case.

### xlsx_add_chart.py
**Purpose:** Add a bar/line/pie chart over a cell range; chart stays editable in Excel/LibreOffice.
