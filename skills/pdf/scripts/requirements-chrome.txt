# Optional dependency for the chrome render engine (--engine chrome).
#
# Install with the bundled Chromium binary via:
#   bash install.sh --with-chrome
# or manually:
#   ./.venv/bin/pip install -r requirements-chrome.txt
#   ./.venv/bin/playwright install chromium
#
# Playwright pulls a self-contained Chromium binary (~150 MB) under
# ~/.cache/ms-playwright (Linux) or ~/Library/Caches/ms-playwright (macOS).
# The binary is platform-specific; redistributing the .venv across
# platforms requires re-running `playwright install chromium` on each.
#
# We pin a major version range, not a release tag — Playwright itself is
# stable and the chromium revision is what changes from release to release.
#
# Floor is 1.42 (not 1.40): Playwright 1.42 added the page.pdf() outline= and
# tagged= options. render_chrome() passes BOTH to emit a PDF outline (TOC
# bookmarks) — Chromium builds the outline from the tagged-PDF structure tree,
# so tagged=True is required alongside outline=True (pdf-7 / TASK 014). On
# 1.40/1.41, page.pdf(outline=True, tagged=True) raises TypeError. install.sh
# --with-chrome installs this file with --upgrade so a re-run lifts a
# pre-existing too-old Playwright to the floor.
playwright>=1.42,<2.0
