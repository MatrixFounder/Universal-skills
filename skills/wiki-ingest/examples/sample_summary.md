## Key Concepts

- **Self-Improving AI Agent** — an agent that closes the loop prompt → strategy → outcome → learning → updated strategy.
- **Four Rules for a Good Trading Agent** — Accuracy, Reliability, Well-Defined Goal, Self-Improvement.
- **Scientific Method for Strategy Iteration** — change exactly one variable, measure, re-baseline.
- **Sharpe Score** — risk-adjusted return: `(R_p − R_f) / σ_p`; min 1 recommended as baseline.
- **Read-Only Review Cycle** — first review run produces a markdown report with no writes until operator approves.
