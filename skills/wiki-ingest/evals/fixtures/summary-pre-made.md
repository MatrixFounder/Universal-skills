# Quant Crypto Handbook ch.4

## TL;DR

> Risk-adjusted return metrics for crypto trading strategies. Covers Sharpe (volatility-adjusted), Sortino (downside-only), and Max Drawdown as a survivability indicator.

## Key Concepts

- **Sharpe Score** — risk-adjusted return measure: `(R_p − R_f) / σ_p`. Many funds prefer Sortino over Sharpe for asymmetric strategies because Sharpe penalises upside volatility too.
- **Sortino Ratio** — like Sharpe but uses downside deviation only. Better for strategies with positive skew.
- **Max Drawdown** — largest peak-to-trough percentage drop. Survivability indicator. Above 50% typically psychologically untradeable.

## Notable Quotes

> "Sharpe is overrated for crypto — try Sortino first."
